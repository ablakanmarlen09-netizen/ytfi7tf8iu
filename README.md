# 📚 Интернет-магазин книг "КнигоМир"

**НЕФУНКЦИОНАЛЬНЫЙ ПРОТОТИП** - Статичные макеты экранов для визуализации требований

## ⚠️ Важно: Это НЕ рабочее приложение!

Это **нефункциональный прототип** - набор статичных макетов экранов, созданных для демонстрации функциональных и нефункциональных требований. Кнопки и формы не работают, это только визуальное представление интерфейса.

## 🎯 Визуализированные экраны

### Для пользователей:
- ✅ Главная страница с баннером и популярными книгами
- ✅ Страница каталога с поиском и фильтрами
- ✅ Детальная страница книги
- ✅ Корзина со статичными товарами
- ✅ Форма оформления заказа
- ✅ Страница входа в систему
- ✅ Страница регистрации

### Для администраторов:
- ✅ Панель управления с статистикой
- ✅ Форма добавления новой книги
- ✅ Страница управления заказами
- ✅ Изменение статусов заказов

## 🚀 Технологии

- **React 18.3** - UI библиотека
- **React Router 7** - навигация между экранами
- **TypeScript** - типизация
- **Tailwind CSS 4** - адаптивная стилизация
- **Lucide React** - иконки

## 📋 Навигация по прототипу

Все ссылки работают и позволяют перемещаться между экранами:

- `/` - Главная страница
- `/catalog` - Каталог книг
- `/book/1` - Детали книги
- `/cart` - Корзина
- `/checkout` - Оформление заказа
- `/login` - Вход
- `/register` - Регистрация
- `/admin` - Админ-панель
- `/admin/add-book` - Добавление книги
- `/admin/orders` - Управление заказами

## 📁 Структура проекта

```
src/app/
├── App.tsx                    # Точка входа с RouterProvider
├── routes.tsx                 # Конфигурация всех маршрутов
├── contexts/
│   └── AppContext.tsx         # Статичные данные для визуализации
├── components/
│   └── Layout.tsx             # Общий шаблон (header, footer)
└── pages/                     # Все экраны прототипа (11 страниц)
    ├── Home.tsx               # Главная с баннером
    ├── Catalog.tsx            # Каталог с поиском
    ├── BookDetails.tsx        # Детали книги
    ├── Cart.tsx               # Корзина
    ├── Checkout.tsx           # Оформление заказа
    ├── Login.tsx              # Вход
    ├── Register.tsx           # Регистрация
    ├── Admin.tsx              # Админ-панель
    ├── AdminAddBook.tsx       # Добавление книги
    ├── AdminOrders.tsx        # Управление заказами
    └── NotFound.tsx           # 404
```

## 🎨 Особенности визуализации

- ✅ **Адаптивный дизайн** - корректное отображение на всех экранах (НФТ)
- ✅ **Интуитивный интерфейс** - понятная навигация и визуальная иерархия (НФТ)
- ✅ **Аннотации требований** - желтые блоки с описанием НФТ на экранах
- ✅ **Статичные данные** - реалистичные демо-книги и заказы
- ✅ **Disabled формы** - поля помечены как макеты

## 📝 Нефункциональные требования в прототипе

Каждое нефункциональное требование визуализировано или задокументировано:

| Требование | Визуализация в прототипе |
|------------|--------------------------|
| **Удобство интерфейса** | Интуитивная навигация, четкая структура экранов |
| **Мобильная адаптивность** | Responsive дизайн через Tailwind CSS |
| **Производительность поиска** | Аннотация на странице каталога |
| **Защита данных** | Аннотации на формах входа/регистрации/оплаты |

## 🔒 Безопасность (документирована)

На экранах Login, Register и Checkout размещены аннотации:

**Что визуализировано:**
- Требования к паролю (минимум 6 символов)
- Проверка формата email
- Админ-панель доступна только администраторам

**Что описано в аннотациях:**
- HTTPS для шифрования трафика
- Bcrypt для хеширования паролей
- JWT токены для аутентификации
- Токенизация платежных данных
